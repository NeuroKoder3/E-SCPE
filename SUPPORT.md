# Support

## Getting Help

Thanks for using E-SCPE! Here's how to get help:

### Documentation

- **[README](README.md)** -- Overview, quickstart, CLI reference, and
  architecture.
- **[Deployment Guide](DEPLOYMENT.md)** -- Installation, configuration,
  licensing, backup/restore, and SIEM integration.
- **[Security Policy](SECURITY.md)** -- Supported versions, security
  architecture, and vulnerability reporting.

### Bug Reports

If you've found a bug, please
[open an issue](../../issues/new?template=bug_report.yml) using the Bug Report
template. Include reproduction steps, expected behavior, and environment
details.

### Feature Requests

Want to suggest an improvement? Please
[open an issue](../../issues/new?template=feature_request.yml) using the
Feature Request template.

### Security Issues

**Do NOT open a public issue for security vulnerabilities.** Please use the
[Security Advisories](../../security/advisories) tab or contact the maintainer
directly. See [SECURITY.md](SECURITY.md) for full details.

### General Questions

For general questions about usage, deployment, or architecture, open an issue
with the `question` label.

## Response Times

This is a single-maintainer project. Response times are best-effort:

| Priority   | Target Response |
|------------|-----------------|
| Security   | 48 hours        |
| Bugs       | 1 week          |
| Features   | 2 weeks         |
| Questions  | 2 weeks         |

## Enterprise Support

For enterprise support, custom integrations, or deployment assistance, please
contact the maintainer through their GitHub profile.
